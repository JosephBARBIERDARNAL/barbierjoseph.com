Natural Language Processing web application


I've created a Streamlit web application that automates a lot of different NLP tasks. Below, you can find an embed version of the app. You can click on the Fullscreen button to see it in another tab.

With it, you can:

  • Apply sentiment analysis to a text
  • Create wordclouds
  • Use regular expressions to find specific elements in a text
  • Measure similarity between two texts

If you're interested in the source code, you can find it on this GitHub repo.
